# Online Shop Customer Frontend

Clean customer-facing frontend skeleton for `online-shop-api`.

## Tech stack

- Next.js
- TypeScript
- Tailwind CSS
- App Router

## Getting started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the local app at:

```text
http://localhost:3000
```

## Checks

Run linting:

```bash
npm run lint
```

Run a production build:

```bash
npm run build
```

## Current scope

This frontend is intentionally small and incremental. Catalog API integration, cart, auth, and checkout are not implemented yet.

The backend API and admin panel are separate parts of the project and are not modified by this customer frontend skeleton.
